#  BookHaven

**Front-End Development Project: Interactive Book Listing**

---

##  Project Overview

This project is a responsive, single-page application built with **React**. It demonstrates core concepts of state management, component separation, and dynamic data handling.

## Key Features

* **Filtering:** Users can filter the book list by genre (Programming, Classics, etc.).
* **Sorting:** Books can be sorted by Publication Year (Newest/Oldest) or Title (A-Z).
* **Searching:** Real-time search by Book Title or Author.
* **Responsive Design:** The layout adapts perfectly to mobile, tablet, and desktop screens.
